# 대진대학교 졸업요건 공식 규칙 조사

> 확인일: 2026-07-11 · 공식 공개 자료만 사용 · 서비스 표시는 `예상 졸업요건 점검`으로 제한

## 결론

- 충돌 회피·공강·시간대 선호만으로 시간표를 생성할 때는 편입 여부가 필요하지 않다. 다만 졸업요건과 남은 필수 이수과목까지 반영하려면 `입학연도`, `현재 학과`, `원 입학 학과`, `학생유형`, `단일·복수·부·마이크로전공`, `편입 여부`가 필요하다.
- 규칙 우선순위는 `학칙·시행세칙 → 해당 입학연도 교육과정편람 → 중앙 졸업심사 XLSX → 최신 학과 공지 → 학과 정적 페이지`로 둔다.
- 현재 학과별 교육과정 46/46과 독립 졸업심사 단위 44/44의 공식 원천을 확보했다. 상세 커버리지는 [학과별 출처 감사](DAEJIN_ACADEMIC_REQUIREMENTS_SOURCE_AUDIT.md)를 따른다.
- 2016~2026 편람 원본과 2026 졸업심사 XLSX·대체/동일과목표를 `data/requirements/raw/`에 저장했다. 2012~2015 편람도 대학 공식 색인과 iBook 원문을 확인했지만 아직 로컬 보존·정규화·검수가 끝나지 않았으므로 해당 학번의 전공필수 충족을 확정하지 않는다.
- 학교 포털의 개인 졸업사정, 상담 승인, 편입 인정학점, 증빙 유효성은 공개 자료로 알 수 없다. 결과는 `SATISFIED`, `REMAINING`, `UNKNOWN`으로 나누고 `UNKNOWN`을 충족으로 간주하지 않는다.

추가 공통 조건:

- 일반 학생은 8학기 이상 등록한다. 조기졸업·편입은 별도 규칙을 적용한다.

## 주요 예외

- 2025 이후 간호학과: 교양 25~42, 전공기초 18 + 전공필수 78 + 전공선택 5로 전공 101학점.
- 미래평생교육융합대학 성인학습자 과정과 국제협력대학 한국학·국제경영 일부 과정: 총 120학점, 교양·전공 기준 별도.
- 편입생: 교양필수는 대순사상과상생윤리 2학점을 기준으로 하며 영역요건을 적용하지 않는다.
- 2025 이후 일반 외국인학생: 교양 32~42, 필수 9 + 선택 23 이상.
- 국제협력대학 외국인학생: 교양 18~42, 선택 18 이상.
- 승인된 장애학생은 교양 영역구분 면제가 가능하지만 교양 총학점이 자동 면제되는 것은 아니다.
- 공학교육인증 참여자는 교양 인정 상한이 최대 50학점일 수 있다.
- 자연·공학 일부 학과는 기초과학 과목·학점 규칙이 별도이므로 학과 규칙으로 저장한다.

## 졸업심사 표준화

2026 중앙 XLSX는 현재·종전·융합을 포함한 66개 단위를 다룬다.

- `A`: 학술 — 논문, 시험, 작품, 보고서
- `C`: 창의·자격 — 자격증, 외국어, 수상
- `E`: 경험·진로 — 취업, 창업, 3개월 이상 인턴, 6~9P 현장실습
- 원문 네 번째 코드는 파일에서 `S`로 표기된 행이 있어 원문 값을 보존한다. 안내 문구의 의미는 봉사·비교과이며, 코드 `V/S` 충돌은 파서가 임의 보정하지 않고 검수 대상으로 둔다.

주전공은 최대 2개 요건군, 복수전공은 최대 1개 요건군을 지정한다. 2026 전환 상태는 표준화 61개, 기존·표준화 병행 3개, 기존 유지 2개다.

원본 파일명은 `v39 (2026.04.15.)`지만 내부 시트는 `version 40 / 2026.04.14`로 표시된다. 두 메타데이터와 원본 SHA-256을 모두 저장하고 `metadata_conflict=true`로 처리한다.

## 자동 판정 금지 항목

다음은 공개 원문만으로 확정할 수 없어 `requires_manual_review=true`가 필요하다.

1. 아직 정규화·검수하지 않은 2012~2025 학과별 전공필수 전체 목록
2. 학과 통폐합·명칭 변경 학생에게 실제 승인된 교육과정
3. 개인별 대체·동일과목·재수강 인정 결과
4. 편입생 인정학점과 학년 배정
5. 공학교육인증 실제 참여·포기 상태
6. 장애학생 면제 승인 여부와 상담 12회 이수기록
7. 학과 자체시험·교수회의 예외 승인
8. 자격증·외국어·공모전·취업·인턴 증빙의 유효성
9. 내부 포털의 최종 졸업사정 결과

## 구현 규칙

- 규칙은 입학연도별 immutable version으로 저장하고 과거 버전을 덮어쓰지 않는다.
- 학과 원문 명칭과 현재 정규화 명칭을 별도 필드로 둔다.
- 모든 결과는 source 문서·페이지/조항·XLSX 시트/행으로 역추적한다.
- 충돌하는 원문은 자동 병합하지 않고 검수 대기 상태로 둔다.
- 시간표 optimizer에는 `검증 완료` 규칙만 hard constraint로 넣고, 미확인 규칙은 경고로만 보여준다.
