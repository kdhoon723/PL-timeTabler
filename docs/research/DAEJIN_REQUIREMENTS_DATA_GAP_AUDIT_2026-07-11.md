# 대진대학교 학년별 필수과목 데이터 공백 감사

> 기준일: 2026-07-11 · 목적: 시간표의 `필수 추천`에 안전하게 자동 반영할 수 있는 범위와 추가 정규화 순서를 확정한다.

## 한눈에 보는 상태

| 범위 | 공식 원문 | 정규화 | 현재 제품 자동 적용 |
|---|---|---|---|
| 2026 입학자 전공필수 | 확보 | 46개 교육과정 단위 상태 보존 | 적용 중 |
| 2025~2016 입학자 전공필수 | 로컬 확보 | 미완료 | 미적용 |
| 2015~2012 입학자 전공필수 | 공식 색인·iBook 확인 | 미완료 | 미적용 |
| 2026 공통 졸업·교양 규칙 | 확보 | 일부 완료 | 예상 점검에 제한 적용 |
| 학과별 졸업평가·자격요건 | 공식 XLSX 확보 | 구조화 완료 | 예상 점검 데이터로 연결 대기 |
| 현재 학기 개설 분반 | 확보 | 카탈로그 정규화 완료 | 적용 중 |

현재 `필수 추천`은 `major-required-courses-2026.json`만 자동 연결한다. 따라서 2026년 3학년처럼 통상 2024 입학 교육과정을 적용받는 학생에게는 공식 2024 원문이 있어도 전공필수를 자동 제시하지 않는다. 이는 원문 부족이 아니라 **정규화·제품 연결 공백**이다.

## 새로 확인한 공식 원천

- [대학생활안내 공식 색인](https://www.daejin.ac.kr/daejin/849/subview.do)
- [2012 공식 iBook](https://ibook.daejin.ac.kr/Viewer/39GPUIGQ510Z)
- [2013 공식 iBook](https://ibook.daejin.ac.kr/Viewer/33O08ODTNCSP)
- [2014 공식 iBook](https://ibook.daejin.ac.kr/Viewer/4CTTQ7GAFUZ7)
- [2015 공식 iBook](https://ibook.daejin.ac.kr/Viewer/3YX8Q3LFBGCI)

공식 뷰어 URL 확인은 자동 판정 가능 상태를 뜻하지 않는다. 원본 파일 또는 페이지 이미지 보존, 학과별 페이지 인덱스, 과목코드·이름·이수구분 추출, 학과 개편 별칭 연결, 사람 검수까지 통과해야 활성화한다.

## 자동으로 미리 채울 수 있는 항목

공식 원문과 학생의 입학연도·학과가 일치하고 검수가 끝난 경우 다음은 미리 채울 수 있다.

1. 입학연도별 전공필수 과목과 학점
2. 졸업 총학점, 교양 최소학점·영역, 전공 이수 경로별 최소학점
3. 학과·전공의 과거 명칭과 현재 명칭 연결
4. 폐지 과목의 동일·대체과목 **후보**
5. 학과별 졸업평가·자격요건 체크리스트
6. 이번 학기에 실제 개설된 필수과목 분반

## 학생 또는 학교 확인이 필요한 항목

다음은 공개 문서만으로 개인별 사실을 확정할 수 없으므로 자동 선택하지 않는다.

- 편입 인정학점과 실제 학년 배정
- 전과·편입·학과 통폐합 학생에게 승인된 적용 교육과정
- 이미 이수했거나 재수강 중인 과목
- 개인별 동일·대체과목 승인 결과
- 복수·부·마이크로전공 승인 상태
- 공학교육인증 참여·포기 상태
- 상담 횟수, 증빙 유효성, 내부 포털의 최종 졸업사정

이 값은 `UNKNOWN`을 기본으로 두고 학생 확인 또는 학교 데이터 연동 후에만 hard constraint로 승격한다.

## 구현 우선순위

1. **2025 → 2024 → 2023 → 2022 정규화:** 현재 재학생의 체감 커버리지를 가장 빠르게 높인다.
2. **공통 교양·졸업 규칙 연결:** 학과와 무관한 규칙을 먼저 재사용한다.
3. **졸업심사 CSV 제품 연결:** 이미 구조화된 학과별 평가 항목을 예상 점검에 노출한다.
4. **2021~2016 정규화:** 복학·졸업유예 학생까지 확장한다.
5. **2015~2012 iBook 수집·추출:** 뷰어 형식에 맞는 별도 수집기를 만든다.

## 추출기 개선 요구

현재 `scripts/extract-major-required.py`는 2026 문서명·페이지 구조와 cohort를 전제로 한다. 연도 공통 파이프라인으로 확장할 때 다음을 분리한다.

- 연도별 source adapter와 cohort 메타데이터
- 학과명 별칭·통폐합 이력
- PDF/HWP/iBook 입력 adapter
- 추출 결과의 합계·중복·코드 검증
- 원문 페이지와 정규화 레코드의 역추적 정보
- 검수 완료 전 비활성화하는 activation 상태

연도별 원문을 단순히 한 파서에 통과시키지 않고, 추출 → 스키마 검증 → diff → 사람 승인 → 활성화 순서를 유지한다.
