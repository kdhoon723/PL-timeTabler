# 팀프로젝트 기능 목록

## 스플래시·온보딩

- 서비스 소개 스플래시 화면
- 학번 입력
- 이름 입력
- 학년 선택
- 학과 선택
- 개인정보 수집·이용 동의
- 회원가입
- 로그인
- 소셜 로그인
- 로그인 세션 유지
- 로그아웃

## 메인 페이지

### 시간표 편성

- 강의 후보를 이용한 시간표 자동 편성
- 원하는 과목 필수 포함
- 원하는 교수 고정
- 원하는 분반 고정
- 자동 생성 시간표 3개 제공
- 자동 생성 시간표별 등교일·학점·공강 시간 비교
- 현재 시간표와 자동 생성 시간표 비교
- 전공 과목 필터링
- 같은 과목의 교수·시간·분반 비교
- 원하는 요일을 공강 요일로 설정
- 특정 요일 수업 완전 제외
- 수업 시작 전·종료 후 시간대 설정
- 수업 사이 공강 최소화
- 수업 사이 허용 공강 시간 설정
- 점심시간 확보
- 하루 최대 수업시간 설정
- 목표·최소·최대 학점 설정
- 시간표에 강의 추가
- 시간표에서 강의 삭제
- 같은 과목의 다른 분반으로 교체
- 시간표 드래그 편집
- 시간 충돌 확인
- 충돌 없는 대체 분반 추천
- 시간표 수정 실행 취소·다시 실행

### 강의 검색

- 과목명 검색
- 교수명 검색
- 과목코드 검색
- 전공 강의 필터
- 교양필수 필터
- 교양선택 필터
- 교양 영역별 필터
- 일반선택 필터
- 학년별 필터
- 요일별 필터
- 강의 이름순 정렬
- 강의 이름 역순 정렬
- 강의 인기순 정렬
- 강의 평점순 정렬
- 리뷰 많은 순 정렬

### 강의 리뷰

- 강의 리뷰 목록 조회
- 과목·교수별 리뷰 조회
- 별점 등록
- 리뷰 작성
- 내가 작성한 리뷰 조회
- 리뷰 수정
- 리뷰 삭제
- 강의별 평균 별점 표시
- 강의별 리뷰 수 표시
- 별점과 리뷰 수를 반영한 인기 점수 계산

### 내가 들은 강의

- 이수한 강의 직접 입력
- 이수 학점 직접 입력
- 전공·교양·영역 정보 직접 입력
- 시간표 또는 성적표 이미지 업로드
- OCR을 이용한 과목명·학점·이수구분 인식
- OCR 인식 결과 확인·수정
- 현재 시간표의 강의를 수강 중 과목으로 등록
- 수강 중 과목을 이수 완료 과목으로 변경
- 이수과목 수정
- 이수과목 삭제
- 부족한 총학점 안내
- 부족한 전공·교양 학점 안내
- 아직 이수하지 않은 교양 영역 안내
- 필수과목 이수 여부 안내
- 이번 학기 수강 가능한 부족 과목 추천

### 시간표 저장·공유

- 시간표 저장
- 시간표 이름 변경
- 저장된 시간표 수정
- 저장된 시간표 삭제
- 시간표 복사
- 시간표 즐겨찾기 등록·해제
- 시간표 공유 링크 생성
- 공유받은 시간표 불러오기
- 시간표 이미지 저장
- 시간표 PDF 출력

## 졸업요건

- 입학연도 선택
- 학과·전공 선택
- 학생 구분 선택
- 전공 방식 선택
- 총 이수학점 확인
- 전공 이수학점 확인
- 교양 이수학점 확인
- 교양 영역별 이수 여부 확인
- 필수과목 이수 여부 확인
- 부족한 학점·과목·영역 안내
- 공식 졸업요건 근거 확인
- 자동 판정이 불가능한 항목 별도 안내

## 마이페이지

- 내 계정정보 조회
- 학번·이름·학년·학과 변경
- 이전 학기 시간표 조회
- 저장한 시간표 조회
- 즐겨찾기 시간표만 조회
- 내가 작성한 리뷰 조회
- 이수과목 조회·수정
- 개인정보 동의 내역 조회
- 로그아웃
- 회원 탈퇴
- 회원 탈퇴 시 사용자 정보·시간표·이수내역·리뷰 삭제

## 세션·접근 제한·오류 처리

- 로그인 세션 만료 안내
- 세션 만료 시 재로그인 안내
- 로그인이 필요한 기능 접근 시 로그인 화면 이동
- 권한이 없는 요청 안내
- 잘못된 입력값 안내
- 존재하지 않는 페이지 안내
- 서버 오류 안내
- 네트워크 연결 오류 안내
- 요청 제한 초과 안내
- 자동 편성 작업 실패·취소·시간 초과 안내

## 시스템 기능

- 학기 목록 조회
- 학기별 강의 데이터 조회
- 강의 데이터 버전 확인
- 자동 편성 작업 생성
- 자동 편성 진행 상태 조회
- 자동 편성 작업 취소
- 서비스 상태 확인
- 사용자별 접근 권한 확인
- 개인정보와 OCR 원본 이미지 삭제 처리
